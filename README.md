# Fortnite Tournament Dashboard

## Descrição
Dashboard interativo desenvolvido com HTML, CSS e JavaScript puro para gerenciar dados de jogadores de um torneio de Fortnite.

## Funcionalidades
Exibição dinâmica de jogadores em cards
Filtro por modo de jogo (Solo, Duos e Squads)
Indicadores automáticos:
  - Total de jogadores
  - Média de K/D Ratio
  - Melhor ranking
Formulário para adicionar novos jogadores
Persistência com localStorage
Layout responsivo com Flexbox

## Estrutura

2. Abra o arquivo index.html no navegador.
